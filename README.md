# ImpactOS AI — Frappe App

Frappe custom app for ImpactOS AI — MIS Blueprint Platform powered by the MAP Framework (Model → Align → Power with AI).

Built by [EdZola Technologies](https://edzola.com).

## Install

```bash
bench get-app https://github.com/akshayedzola/impact-os-ai-frappe
bench --site yoursite.local install-app impact_os_ai
```
